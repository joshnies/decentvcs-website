# DecentVCS Website

Website for DecentVCS.

## Requirements

| Dependency                     | Version |
| ------------------------------ | ------- |
| [Node.js](https://nodejs.org/) | >= 19   |
| [pnpm](https://pnpm.io/)       | >= 7    |

## Installation

```bash
pnpm install
```

## Building

> Only required for production builds.

```bash
pnpm build
```

## Running

```bash
# development
pnpm start

# production
pnpm serve
```
